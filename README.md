# fundamentals-of-python

Python Main Task:
Market System Build a market system in Python that simulates a cashier's functionality. The system should include the following features:

Warehouse Management: Create a dictionary to store product information, including:
Product name. Price (in AZN). Stock quantity.

Cashier Functionality: Display all available products, their prices, and stock quantities.
Allow the user to select a product to purchase. Ask the user for the quantity they want to buy. Check if the requested quantity is available in the stock: If enough stock is available, reduce the stock by the requested amount and add the product to the shopping cart. If not enough stock is available, display an error message and ask the user to choose again.

Cart and Payment: ○ Calculate the total price of all products in the shopping cart. ○ Print a receipt at the end of the transaction, showing:
The products purchased. The quantity of each product. The price for each product and the total amount.

Exit Option: Allow the user to exit the system by typing 'q'.
